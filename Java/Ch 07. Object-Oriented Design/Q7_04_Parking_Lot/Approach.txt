ParkingLot has
  - Levels
  
Each Level has
  - ParkingSpot
  - Level/floor number
  - No of free spots
  
Each ParkingSpot has 
  - Vehicle
  - VehicleSize
  - Level
  
Each Vehicle has
  - VehicleSize
  - Licence plate string
  - No of spots needed to park (e.g Bus requires 5)
  
Each Vehicle can either by
  - Motorcycle
  - Bus
  - Car
